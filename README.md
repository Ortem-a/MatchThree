# MatchThree
 Техническое задание для МЕДИА-ТЕЛЕКОМА

## Задание
 Необходимо разработать игру жанра "3 в ряд".
 Игроку отводится на матч 60 секунд. Дальше игра завершается.
 
## Управление
 Передвижение фигур просиходит с помощью свайпов мышью.
